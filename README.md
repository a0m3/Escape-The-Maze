# Escape the Maze

## Technologies Used
- HTML 
- CSS 
- JavaScript

## Description

Maze Escape Game is a simple browser game built using HTML, CSS, and JavaScript. The goal is to move through different maze levels and reach the exit while avoiding walls and completing each level as fast as possible.

The game has 5 levels with increasing difficulty, a timer for each level, and a light/dark mode option in the settings to improve the player experience.

## User Stories

1. As a player, I want to see a main menu so I can start or configure the game easily.
2. As a player, I want to change the color theme from light to dark.
2. As a player, I want to start the game from a clear start button.
3. As a player, I want to navigate through a maze so I can try to reach the exit.
4. As a player, I want to move my character using keyboard controls.
5. As a player, I want to avoid walls so that the maze feels challenging.
6. As a player, I want to reach the exit to complete a level before the time finishes.
7. As a player, I want to get a menu includes next level or restart level.
